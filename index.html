Command & Conquer: Generals Modern

Generals Modern is an ambitious overhaul of the C&C: Generals Zero Hour engine, focused on modernization, resolution fixes, and new rendering systems using BGFX.

Engine Upgrade Overview

This project removes the legacy DirectX 8 rendering backend and replaces it with a modern BGFX-based rendering system. The goal is to fully support 1440p/4K resolutions, scalable UI, smoother camera behavior, and cross-platform GPU support.

We're carefully refactoring the old engine to:

  • Support high-resolution rendering via BGFX
  • Replace outdated DX8 pipeline with shaderc-compiled GLSL/HLSL shaders
  • Fix broken fullscreen and input behavior
  • Enable new UI layouts and map mechanics
